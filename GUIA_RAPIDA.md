# 🚀 Guía Rápida de Instalación - LabelFresh

## Instalación en 3 pasos (Windows, Mac, Linux)

### Paso 1: Descargar Node.js
- Ve a: https://nodejs.org/
- Descarga la versión LTS (recomendada)
- Instala Node.js con las opciones por defecto

### Paso 2: Descargar LabelFresh
- Ve a: https://github.com/huvgalindo-blip/labelfresh
- Haz clic en el botón verde "Code"
- Selecciona "Download ZIP"
- Descomprime el archivo en tu escritorio

### Paso 3: Instalar y ejecutar
1. Abre la carpeta descomprimida
2. **Windows:** Haz clic en la barra de dirección, escribe `cmd` y presiona Enter
3. **Mac/Linux:** Abre terminal en esa carpeta
4. Escribe estos comandos uno por uno:

```bash
npm install
npm start
```

5. Abre tu navegador y ve a: `http://localhost:3000`

## ⏱️ Tiempos estimados:
- Descargar Node.js: 5-10 minutos
- Descargar LabelFresh: 1-2 minutos
- `npm install`: 5-10 minutos (primera vez)
- `npm start`: 5 segundos

## ❓ ¿Necesitas ayuda?
- Revisa el README.md para instrucciones detalladas
- Asegúrate de tener conexión a internet durante la instalación
- Si `npm install` falla, inténtalo de nuevo

## 🎯 ¡Listo!
Una vez que veas el mensaje "LabelFresh - Generador de Etiquetas" en la terminal, la aplicación está lista para usar.
